Designer Quotes

I decited to create a website for my favorite quotes said by some of the greates designers. Also I wanted to sharpen my web development skills.
I used a JS templete from codedrops to create the beautiful image scroll effect you can see on the website.

Role: Web Designer, Developer

FOCUS 01 Website design

Small typography contrasting with the large typography and large margins give the overall experience a refined elegance that’s just as sophisticated as the designers themselfs. The scrolling effect adds to the expirience a silk smooth slide through the website.

FOCUS 02 Responsive Design

Whenever I read something on most websites I find that the font size is too small, so I carefully optimize the font size and line height for mobile to make it bigger and more readable, as that is the whole point of the website.
With Figma Mirror and "Inspect" in the browser it was a piece of cake.
